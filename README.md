# 🎲 Physics Dice Roller

A web-based physics simulation of dice rolling with realistic 3D graphics and physics.

## Features

- 🎲 Realistic physics simulation using Cannon.js
- 🎨 3D rendering with Three.js
- 🎯 Interactive mouse/touch controls
- 🎲 Multiple dice types (4, 6, 8, 10, 12, 20 sides)
- 📱 Mobile-friendly touch support
- 🎭 Realistic lighting and shadows

## How to Use

1. Open `index.html` in a web browser
2. Enter the number of sides (2-20) in the input field
3. Click "Roll Dice" or click/drag on the dice
4. Watch the physics simulation and see your result!

## Controls

- **Click/Spark**: Click on the dice or the "Roll Dice" button to roll
- **Mouse Drag**: Grab and throw the dice with your mouse
- **Spacebar**: Quick roll shortcut
- **Touch**: Tap or drag on mobile devices

## Technical Details

- **Physics Engine**: Cannon.js for realistic physics simulation
- **3D Rendering**: Three.js for WebGL graphics
- **Responsive**: Adapts to any screen size
- **Cross-platform**: Works on desktop and mobile devices
